Video Editor Telegram Bot

بوت تيليجرام لتعديل الفيديوهات.

المميزات

- استقبال الفيديو من المستخدم.
- إضافة نصوص على الفيديو.
- إضافة موسيقى.
- تغيير سرعة الفيديو.
- إضافة شعار.
- لوحة تحكم للمشرف.
- إمكانية فتح أو إغلاق الميزات.

التثبيت

pip install -r requirements.txt

التشغيل

python bot.py

الإعداد

افتح ملف "config.py" وأضف:

- توكن البوت.
- ID المشرف.

الملفات

- bot.py
- config.py
- admin.py
- video_editor.py
- features.json

المطور

تم إنشاء هذا المشروع باستخدام Python و Telegram Bot API.
